# Liane’s Library

Liane’s Library is a simple Streamlit app to track books, friends, and borrowed books.

## What this project does
This app allows you to:
- Add books
- Add friends
- Borrow books
- Return books
- Track who borrowed what
- Delete books, friends, and loan records

## Tech Stack
- Python
- Streamlit
- SQLAlchemy
- SQLite
